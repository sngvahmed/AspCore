# Response Guide

Http status code 200
` return Ok(); `

Http status code 201
` return Created(); ` 

Http status code 204
` return NoContent(); `

Http status code 400
` return BadRequest(); `

Http status code 401
` return Unauthorized(); ` 

Http status code 403
` return Forbid(); ` 

Http status code 404
` return NotFound(); `
